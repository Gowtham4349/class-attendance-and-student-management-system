# 🎓 Student Performance Data Analysis

A professional data analysis project that explores how attendance, study hours, and assignment scores affect students' academic performance. The project uses Python, SQL, and Power BI to clean, analyze, visualize, and generate meaningful educational insights.

---

## 📌 Project Overview

Educational institutions collect large amounts of student data, but converting it into actionable insights can be challenging. This project analyzes student performance data to identify the key factors influencing exam scores and supports data-driven academic decisions.

---

## 🎯 Objectives

- Analyze student academic performance.
- Identify factors affecting exam scores.
- Measure the impact of attendance and study hours.
- Discover high-performing students.
- Generate business insights for educational institutions.

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/student-performance-data-analysis.git
```

### 2. Navigate to the project

```bash
cd student-performance-data-analysis
```

### 3. Install dependencies

```bash

```

```bash
python src/data_cleaning.py
```
